### Action Detail
		
<a name="SaveAsFile"></a>    
#### SaveAsFile(fileName, format)

Saves the screenshot to a file.


**Parameters:**

|	**Name** | **Type** | **Description** |
| ---------- | -------- | --------------- |
| fileName | string |	Path to a file |
| format | string |	Format of the image, one of bmp, emf, exif, gif, icon, jpeg, png, tiff, wmf |
