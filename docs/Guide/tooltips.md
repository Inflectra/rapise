# Tooltips

## Purpose

**Tooltips** let you view a variable's value during debugging.

## Usage

1. Put a breakpoint in the script at or near where you wish to investigate
2. Mouse over variables as you advance through the script. Â  A small box will popup, displaying the variables' values:
